We distribute these components in two ways:

Open source code

Both the BabyLogger schematics and code, and the modules of the ResearcherPortal are open source and can be reproduced and modified (MIT open source license, see our GitHub). Contributors are welcome.

Ready to use devices and web service

We have setup a non-profit (BabyCloud-NP) which distributes at the lowest possible cost, but without putting a non-sustainable burden on the research labs contributing to this project, the BabyCloud components for the non-technical savy behavioral scientist. It helps researchers group order BabyLoggers in order to lower the construction price, gives them access to an on-line version of the ResearcherPortal running on a local GDPR-compliant cloud in Europe, and some customer support.
