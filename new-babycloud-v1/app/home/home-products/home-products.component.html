Baby Logger

A small, user-friendly, baby-compatible, wearable audio recorder. It has 4 channels, an accelerometer, 24 hours of recording autonomy, stores up to 120h of data and is encrypted. With the logger comes a set of tools to extract and decrypt the data.

Researcher Portal

A cloud platform for annotating, analyzing and archiving audio data with secure state-of-the-art machine learning tools (signal processing, speech recognition). The data is stored encrypted, the analysis is performed in the cloud itself and can be securely shared and analyzed in situ insuring end-to-end privacy.
